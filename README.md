¡Bienvenido a mi portafolio profesional! Soy **Sergio Villena Vergara**, Ingeniero Civil en Informática especializado en desarrollo full-stack, automatización avanzada con Inteligencia Artificial, sistemas de información geográfica (GIS) e infraestructura autohospedada (self-hosted).

---

## 🛠️ Tecnologías y Herramientas

El portafolio está construido con una arquitectura moderna que prioriza el rendimiento extremo, la accesibilidad y el SEO:

- **Core & Renderizado**: [Astro](https://astro.build/) (generación de sitios estáticos para velocidad ultra-rápida y SEO óptimo).
- **Componentes Interactivos**: [React](https://react.dev/) + [Framer Motion](https://www.framer.com/motion/) para transiciones fluidas e interacciones premium.
- **Efectos e Interactividad**: [GSAP](https://gsap.com/) para efectos de micro-interacción avanzados (como el cursor magnético).
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/) para un diseño responsivo, moderno y limpio.
- **Ecosistema**: [n8n](https://n8n.io/), [Docker](https://www.docker.com/), [PostGIS](https://postgis.net/) y [Proxmox VE](https://www.proxmox.com/) para infraestructura y flujos de automatización integrados.

---

## 🚀 Características Principales

1. **Rendimiento Ultra-Rápido**: Renderizado estático con Astro, optimización y compresión exhaustiva de imágenes en formato moderno (`.webp`).
2. **Experiencia de Usuario Premium**: Micro-animaciones cuidadas, scroll interactivo y cursor magnético reactivo.
3. **Diseño Responsivo Completo**: Optimizado para dispositivos móviles, tablets y pantallas de escritorio grandes con navegación adaptada y control de scroll avanzado en modales de proyectos.
4. **Optimización SEO y Semántica**: Estructura HTML5 semántica, tags Open Graph y Twitter Cards completos para previsualización enriquecida en redes sociales, enlaces canónicos automáticos y datos estructurados JSON-LD (Schema.org Person) validados.
5. **Accesibilidad Integrada (A11y)**: Etiquetas `aria-label` descriptivas, soporte para lectores de pantalla, gestión del foco del teclado en diálogos interactivos y ocultamiento de iconos decorativos (`aria-hidden`).

## 💻 Desarrollo e Instalación

Para ejecutar este proyecto de forma local en tu máquina, sigue estos pasos:

### Prerrequisitos
Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior recomendada).

### 1. Clonar el repositorio
```bash
git clone https://github.com/SAVillena/Portafolio.git
cd Portafolio
```

### 2. Instalar dependencias
```bash
npm install
```

### 3. Iniciar el servidor de desarrollo
```bash
npm run dev
```
El servidor se iniciará en `http://localhost:4321`.

### 4. Construir para producción
Para compilar y minificar el sitio para su despliegue final:
```bash
npm run build
```
Los archivos optimizados y listos para producción se generarán en la carpeta `dist/`.

### 5. Validar calidad de código y linter
```bash
npm run lint
```
